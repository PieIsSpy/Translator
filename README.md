# Translator
This is a simple program that manages entries of Language-Word pairs and translates text inputs or files from one language to another.

Originally finished in 3/31/2025

### Features
- Managing Entries
  - Adding and Deleting Entries of Language-Word pairs
  - Adding and Deleting Language-Word pairs in Entries
  - Searching for a word and its translations in entries
  - Importing and Exporting Entries from and to text files
- Translating
  - Translating text inputs from one language to another given a source entries file
  - Translating text files from one language to another given a source text file and a source entries file
    - Also exports the translation into a text file

### Prerequisites
In order to be able to run this program, you will need to install the `gcc` compiler

### Compiling
To compile the project, run:
```
gcc Main.c -o Main.exe
```

### Running
Run `Main.exe` file
